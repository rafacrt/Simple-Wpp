A simple floating WhatsApp button for WordPress. Easy to use and customize!

== Description ==

Simple Wpp is a lightweight plugin that adds a floating WhatsApp button to your WordPress site. With this plugin, users can contact you directly via WhatsApp with just one click.

Features:
* Floating WhatsApp icon
* Customizable number and message
* Works on desktop and mobile
* Lightweight and fast

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory or install via WordPress admin panel.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to **Settings > Simple Wpp** to configure your WhatsApp number and welcome message.

== Frequently Asked Questions ==

= Can I customize the WhatsApp message? =
Yes! You can enter a default message that users will send when clicking the button.

= Can I change the position of the button? =
Currently, the plugin places the button in the bottom-right corner. More options will be available in future updates.
